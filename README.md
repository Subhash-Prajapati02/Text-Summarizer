# 🤖 AI Text Summarizer

An AI-powered web application that generates concise summaries from long text using HuggingFace Transformers (T5) and FastAPI.

---

## 🚀 Features

- ✨ Generate summaries from long text instantly  
- ⚡ FastAPI backend for high performance  
- 🧠 Powered by HuggingFace Transformers (T5 model)  
- 🎨 Clean and modern UI  
- 🔄 Real-time summarization (no page reload)  
- 🌐 Ready for deployment  

---

## 🛠️ Tech Stack

- **Backend:** FastAPI  
- **Frontend:** HTML, CSS, JavaScript  
- **ML/NLP:** HuggingFace Transformers, PyTorch  
- **Model:** T5 (Text-to-Text Transfer Transformer)
